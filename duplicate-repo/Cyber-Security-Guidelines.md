# Cyber Security Guidelines

## Research & Learn
### Common cyber security threats in remote work

Remote work environments can expose companies to several cyber security risks. 
Common threats include phishing emails, insecure Wi-Fi networks, malware, and unauthorized access to company systems. 
Attackers may attempt to trick employees into revealing sensitive information or clicking malicious links.

### Best practices for device and account security

To keep devices and accounts secure, it is important to:

- Use strong and unique passwords
- Enable two-factor authentication (2FA)
- Keep operating systems and software updated
- Avoid connecting to unsecured public Wi-Fi
- Use trusted password managers
- Lock devices when not in use

### Importance of locking your computer

Locking your computer prevents unauthorized access to company data. 
If a computer is left unlocked, anyone nearby could access sensitive information, install malicious software, or misuse company systems.

### Handling phishing attempts

If I receive a suspicious email or link, I will:

- Avoid clicking unknown links
- Verify the sender’s email address
- Check for unusual requests such as urgent payments or password resets
- Report the email to the appropriate team or mark it as phishing

### Strong passwords and password managers

A strong password should:

- Be at least 12 characters long
- Include uppercase letters, lowercase letters, numbers, and symbols
- Be unique for each account

Using a password manager helps securely store passwords and prevents the need to reuse passwords across multiple services.

### Importance of Two-Factor Authentication (2FA)

Two-factor authentication adds an extra layer of security by requiring a second verification step, such as a mobile authentication code. 
Even if a password is compromised, attackers cannot access the account without the second factor.
