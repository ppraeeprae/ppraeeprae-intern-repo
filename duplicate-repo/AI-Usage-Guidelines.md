# Use of AI Tools

## Research & Learn
### AI tools used in my role

As a Website Developer Intern, AI tools can help with several tasks such as generating example code, explaining programming concepts, improving documentation, and suggesting improvements for website content. Tools like ChatGPT can assist with debugging code, understanding technical topics, and drafting documentation more efficiently.

### Benefits and Risks of using AI in a professional setting

AI tools can improve productivity by helping developers find solutions faster, generate ideas, and assist with writing code or documentation. They can also help explain complex concepts in a simple way.

However, there are also risks when using AI in a professional environment. AI-generated responses may sometimes be incorrect or outdated. Another risk is accidentally sharing sensitive company information with AI tools. Because of this, AI should always be used carefully and its output should be reviewed before using it.

### Information that should never be entered into AI tools

Some types of information should never be shared with AI tools, including:

- Confidential company data
- Personal user information
- Internal company documents
- API keys, passwords, or authentication tokens
- Sensitive project details

Keeping this information private helps protect both the company and its users.

### Validating AI-Generated content

AI-generated content should always be reviewed before it is used. To ensure accuracy, I will:

- Carefully read and review the AI-generated response
- Compare the information with official documentation
- Test any AI-generated code before using it
- Edit and improve the content if necessary

This ensures the final output is reliable and suitable for professional use.

## Reflection
### When to use AI vs My own skills

AI can be helpful when learning new concepts, exploring solutions, or improving productivity. For example, I can use AI to understand programming concepts or generate ideas for solving coding problems.

However, I should still rely on my own understanding when writing code or making important technical decisions. AI should support my learning rather than replace my own problem-solving skills.

### Avoiding over-reliance on AI

To avoid relying too heavily on AI, I will first try to solve problems on my own before asking AI for help. If I use AI to generate code or explanations, I will make sure I understand how the solution works instead of simply copying it.

### Protecting data privacy when using AI

When using AI tools, I will avoid including any confidential company information, personal data, or sensitive project details in my prompts. I will only provide general information that is safe to share.
