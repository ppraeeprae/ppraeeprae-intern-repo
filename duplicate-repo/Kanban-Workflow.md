# Agile Workflows & Kanban

## Research & Learn

### How a Kanban board works
A Kanban board is a visual tool used to manage tasks and track progress in a workflow. Tasks are usually represented as cards that move across different columns representing stages of work. This helps the team clearly see what tasks are waiting, what tasks are currently being worked on, and what tasks are completed.

### Kanban Columns
A typical Kanban board may include the following columns:

**Backlog / Not Started**  
This column contains tasks that have been planned but have not yet been started.

**In Progress**  
Tasks that are currently being worked on by a team member.

**Blocked**  
Tasks that cannot continue due to an issue, dependency, or missing information.

**Ready for Review**  
Tasks that are completed and waiting to be reviewed or tested.

**Done**  
Tasks that have been fully completed and approved.

### Task Movement
Tasks move from left to right across the board as progress is made. Usually, the person working on the task is responsible for updating the task status to reflect the current stage. This helps keep the board accurate and allows the team to track progress easily.

### Limiting Work in Progress (WIP)
Limiting Work in Progress means that team members should only work on a small number of tasks at the same time. This helps prevent overload and ensures tasks are completed before new ones are started. It also helps improve focus and productivity.

## Reflection

### How Kanban helps manage priorities
Kanban helps manage priorities by clearly showing which tasks need to be worked on first. By visualizing tasks on the board, it becomes easier to see what needs attention and what is already in progress. It also helps avoid taking on too many tasks at once.

### Improving Mm workflow using Kanban
Using Kanban principles can help me stay more organized with my tasks. By focusing on completing one task before starting another, I can maintain better productivity. Updating task status regularly also helps keep the team informed about progress.
